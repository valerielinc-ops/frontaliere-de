Rollenüberblick

Hintergrund und Mission Swissnoso, das nationale Zentrum zur Prävention von Nosocomial-Infektionen, wurde seit 2009 beauftragt, chirurgische Standortinfektionen (SSI) in Schweizer Krankenhäusern zu überwachen.

Epidemiologische Überwachung von ISS ist eine prospektive Methode zur Überwachung von Patienten, die zur Erkennung und Diagnose von Infektionen mit einer validierten Methode auf Basis internationaler klinischer Kriterien betrieben werden.

Es ist eine ständige Tätigkeit für die Erhebung, Analyse, Interpretation von Daten und die Rückgabe von Ergebnissen Die Krankenschwester, die für die Überwachung von chirurgischen Standortinfektionen verantwortlich ist, sorgt für eine aktive Nachverfolgung von postoperativen Patienten, um Infektionen mit einer validierten Methode basierend auf klinischen und internationalen Kriterien zu erkennen.
